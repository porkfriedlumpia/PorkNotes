# CaramelNotes

Write notes about other players.

A plugin for Vanilla WoW (1.12) or WotLK (3.3.5).
Compatible with TurtleWoW and Project Epoch.

Heavily inspired by [Character Notes](https://www.curseforge.com/wow/addons/character-notes).

## Downloads

#### URL for addon managers

- Vanilla WoW (1.12) : https://github.com/MrToffee/CaramelNotes
- WotLK (3.3.5) : https://github.com/MrToffee/CaramelNotes-WotLK

#### Manual installation

- Vanilla WoW (1.12) : [CaramelNotes-main.zip](https://github.com/MrToffee/CaramelNotes/archive/refs/heads/main.zip) (rename the folder to `CaramelNotes`!)
- WotLK (3.3.5) : [CaramelNotes-WotLK-main.zip](https://github.com/MrToffee/CaramelNotes-WotLK/archive/refs/heads/main.zip) (rename the folder to `CaramelNotes-WotLK`!)
